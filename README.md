# nodejs-CRUD-app

# Description
CRUD application made with NodeJs, Express and MongoDb (Task solution for Zuri Internship)

# Routes

## CREATE DATA ENTRY
POST request to '/create' route

## GET DATA ENTRIES
GET request to '/' route

## UPDATE A DATA ENTRY
PUT request to '/people/:id' route

## DELETE A DATA ENTRY
DELETE request to '/people/:id' route
